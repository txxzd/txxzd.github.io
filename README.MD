# Portfolio Website

This is a simple static portfolio website with a retro terminal-inspired design.

## Files
- `index.html` — main page structure and portfolio content.
- `styles.css` — theme styling, layout, and responsive rules.
- `script.js` — mobile menu toggle and smooth navigation behavior.

## Preview
Open `index.html` in your browser to view the portfolio.

## Customize
- Replace the placeholder experience and skills with your actual resume summary.
- Update the project cards with real project names, descriptions, tech stacks, and links.
- Change the contact email and any personal text in the About section.

## Deployment
This is a static site and can be deployed using GitHub Pages, Netlify, Vercel, or any static hosting.
